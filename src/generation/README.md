# Generation Module

This module contains scripts used to generate synthetic data using the trained generative models.

## Files

- `diffusion.py`: Generates synthetic MedMNIST data using the Conditional Diffusion Model.
- `gan.py`: Generates synthetic data using the GAN Generator.
- `latent_diffusion.py`: Generates synthetic data via Latent Diffusion and VAE Decoder.
- `uncond.py`: Generates unconditional baseline data as `.png` samples or `.npz` datasets.
