# Utilities Module

This module contains helper scripts and classes for various shared tasks across the pipeline.

## Files

- `eda.py`: Contains functions for Exploratory Data Analysis (EDA) on the dataset splits.
- `eval_fid.py`: Calculates Frechet Inception Distance (FID) to evaluate the quality of generated images.
- `logger.py`: Custom logging classes and setup functions for tracking ETL processes and training runs.
- `summarize.py`: Summarizes evaluation metrics from various text files into a unified Markdown table.
